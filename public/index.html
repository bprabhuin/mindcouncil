Frequently asked questions
What is Council Mode?+
Council Mode runs three AI agents — Visionary, Analyst, and User Advocate — through a 4-round structured debate. Each agent generates ideas, critiques the others, refines positions, and scores the final answer. The result is a synthesized verdict with full reasoning transparency.
Can I cancel anytime?+
Yes. Cancel any time from Settings. Your plan stays active until the end of the billing period, then you return to the Free tier. No hidden fees or cancellation charges.
How does API access work?+
Team plan members get a REST API key to trigger Council sessions programmatically, stream debate rounds into their own UI, and integrate verdicts into external tools and workflows.
What AI models power the agents?+
All agents are powered by Claude (Anthropic). Each receives a distinct system prompt shaping its role and reasoning style. Our orchestration layer manages parallel execution, debate sequencing, and scoring entirely on the server — your API key is never exposed to the browser.
